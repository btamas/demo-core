# demo-core

This is a POC of the new bundling system for oat-sa core libraries.

## Build instruction
- Requirements
    - node
    - npm

- Install modules
    `npm i`

- Build modules
    - With webpack: `npm run build`
    - With rollup: `npm run build:rollup`

- Using modules
    - Libraries are built under `dist/js` directory in UMD format.
